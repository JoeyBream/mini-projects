# Joey's Mini Projects Collection

A collection of 11 quick, interactive web applications built with HTML, CSS, and JavaScript.

## Projects

1. **Snake Game** - Classic snake game with arrow key controls
2. **Todo List App** - Task manager with local storage and filters
3. **Weather Dashboard** - Real-time weather information (demo mode)
4. **Password Generator** - Secure password creation tool with strength meter
5. **Expense Tracker** - Personal finance manager with categories
6. **Pomodoro Timer** - Productivity timer with session tracking
7. **Quiz App** - Interactive trivia game with 10 questions
8. **Drawing App** - Digital canvas for creativity with brush and eraser
9. **Music Player** - Custom audio player interface with playlist
10. **Chat Bot** - Interactive conversational bot with personality
11. **Data Table App** - Employee database with search, filter, and sort

## How to Use

Each project is in its own directory. Simply open the `index.html` file in your web browser to run any project.

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript (no frameworks required)
